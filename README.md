# ISE – CGPA / SGPA Calculator

A fully responsive, mobile-friendly **CGPA & SGPA calculator** built for  
**Information Science & Engineering (ISE)** students of **WEC – Puducherry**.

# Live Demo
[https://pavithrab-wec.github.io/wec-ise-cgpa-calculator/](https://pavithrab-wec.github.io/wec-ise-cgpa-calculator/)

This tool helps students calculate:

- Semester-wise **SGPA**
- Overall **CGPA**
- Convert CGPA → Percentage
- Download results as **PDF report**
- Auto-save SGPA inputs
- Light / Dark Mode Theme

---

## 🚀 Features

### 🎓 **SGPA Calculator**
- Select any semester (1–8)
- Auto-loads subjects & credit points
- Grade options follow PTU guidelines (S, A, B, C, D, E, F, FA)
- Alerts you if any subject grade is missing
- Instant SGPA calculation

### 🧮 **CGPA Calculator**
- Enter SGPA for any semester (fill only what is needed)
- Automatically calculates CGPA
- Can convert CGPA → percentage

### 🌓 **Dark / Light Mode**
- Modern UI with purple gradient theme
- Toggle button for dark mode

### 📝 **PDF Report Download**
- Generates printable PDF containing:
  - All SGPA values
  - Final CGPA
  - Percentage

### 💾 **Auto Save**
- CGPA & SGPA inputs are saved in browser local storage
- Values stay even after refreshing the page

---
## 🛠️ **Tech Stack**
- **HTML5**
- **CSS3**
- **JavaScript**
- **LocalStorage**
- **Window Print API** for PDF export
